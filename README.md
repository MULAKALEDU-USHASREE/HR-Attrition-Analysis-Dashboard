# HR-Attrition-Analysis-Dashboard


## Overview

This project analyzes employee attrition using the IBM HR Analytics dataset. The dashboard provides HR insights through interactive visualizations built in Power BI.

## Features

- KPI Cards
  - Total Employees
  - Attrited Employees
  - Attrition Rate
  - Average Monthly Income
  - Overtime Attrition Rate

- Interactive slicers
- Department-wise attrition analysis
- Job role analysis
- Gender distribution
- Job satisfaction analysis
- Age distribution
- Education field analysis

## Tools Used

- Power BI Desktop
- Power Query
- DAX

## Dataset

IBM HR Analytics Employee Attrition Dataset

## Key Insights

- Research & Development has the highest attrition.
- Employees working overtime have a higher attrition rate.
- Job satisfaction levels influence employee retention.
- Life Sciences has the highest attrition among education fields.
